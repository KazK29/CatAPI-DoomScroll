A web page that shows random cat images.  

## Features
- Loads a random cat image on page load
- Click the "New Cat" button to load another cat
- Infinite scrolling: new cats appear as you scroll down
- Simple, clean design with styled images

## How to Use
1. Open `index.html` in your browser
2. Click **New Cat** to load a random cat
3. Scroll down to load more cats automatically
